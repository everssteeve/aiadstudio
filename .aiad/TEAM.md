# Équipe AIAD Studio — Phase Bootstrap

| Responsabilité | Titulaire | Notes |
|----------------|-----------|-------|
| **PM** | Steeve Evers | Cadrage produit, validation Intents, PRD |
| **PE** | Steeve Evers | Rédaction Intents/SPECs, orchestration agents, Drift Lock |
| **AE** | Steeve Evers | Configuration agents, AGENT-GUIDE, Context Budget |
| **QA** | Steeve Evers | Validation Gate, tests, gouvernance |
| **TL** | Steeve Evers | Architecture, ADRs, conventions |

> En phase bootstrap solo, une même personne porte toutes les casquettes.
> L'important est de **changer de casquette consciemment** à chaque étape du cycle SDD.
> Quand tu rédiges un Intent, tu es PM/PE.
> Quand tu passes le Gate, tu es QA.
> Quand tu reviews l'architecture, tu es TL.
